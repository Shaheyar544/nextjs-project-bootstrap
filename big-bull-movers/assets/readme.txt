This folder contains all images and icons used in the Big Bull Movers WordPress theme.

Please add the following images here:
- logo.png (the Big Bull Movers logo)
- truck.png (image of the truck used in the hero section)
- movers.jpg (image of movers for the about section)
- icons/reliable.svg (icon for reliable movers)
- icons/pricing.svg (icon for pricing)
- icons/fully-equipped.svg (icon for fully equipped services)

You can replace these placeholder images with your own or download similar free images/icons.

Make sure to keep the folder structure as:
assets/
  images/
    logo.png
    truck.png
    movers.jpg
  icons/
    reliable.svg
    pricing.svg
    fully-equipped.svg
